===================
Module 1-my_list.py
===================
This module defines class that inherits from list.

Usage
=====
    "class MyList(list)" is used to add new methods to the list class,
    from which it inherates.

::
    >>> MyList = __import__('1-my_list').MyList
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Tests
=====
Normal functionality: my_list instanciates a new list object, can print sorted lists, inherits from list
so all list methods should work as expected.

::
    >>> my_list = MyList([1, 2, 3, 4])
    >>> type(my_list)
    <class '1-my_list.MyList'>
    >>> print(my_list)
    [1, 2, 3, 4]
    >>> my_list[3] = 5
    >>> print(my_list)
    [1, 2, 3, 5]
    >>> my_list.remove(5)
    >>> print(my_list)
    [1, 2, 3]

::
    >>> my_list = MyList()
    >>> print(my_list)
    []
    >>> my_list.print_sorted()
    []

::
    >>> my_list = MyList()
    >>> my_list.append(89)
    >>> my_list.append(-45)
    >>> my_list.append(5)
    >>> print(my_list)
    [89, -45, 5]
    >>> my_list.print_sorted()
    [-45, 5, 89]
